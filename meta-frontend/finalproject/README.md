# Little Lemon Table Reservation System

## Project Overview
This is the Capstone project for the Meta Front-End Developer Professional Certificate. It implements a table booking system for the Little Lemon restaurant.

## Features
- **Dynamic Time Slots:** Times update based on date selection using `useReducer`.
- **Accessibility:** Fully semantic HTML with ARIA labels.
- **Form Validation:** Client-side validation for reservations.
- **Unit Testing:** Verified with Jest and React Testing Library.

## How to Install and Run
1. Clone the repository: `git clone [YOUR_GITHUB_URL]`
2. Run `npm install` to install dependencies.
3. Run `npm start` to launch the app.
4. Run `npm test` to verify unit tests.
